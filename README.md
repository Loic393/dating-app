# Dating Application

Development of an Android app

## Principle of the App

The goal of this project is to create an Android application inspired by dating apps. The majority of the features present in these apps are implemented.

## App Features

- User login/registration and Profile setup.
- View and swipe other user's profile.
- Match and chat with other users.
- Filter users base on a range of age and height.
- FAQs section for user support.

Added features since project demo:
- Matches and chat activity.
- FAQs section.
- User filtering (base on min/max height and age).
- Login with Google Account.
- User profile modification (change height and biography).
- UX/UI improvements.

## Technology used

- Firebase Authentication
- Firestore Database
- Firebase Storage
- Google Authentication
- Yuyakaido CardStackView
- And other dependencies
